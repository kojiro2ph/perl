MakeStrSETFromHashWithFilter --- 連想配列を SET 文字列にする（フィルターつき）

渡し値(integer)mode:オプション
(string array)remove:SETしないフィールドの配列
(hash)hash:SETする連想配列
戻り値SET k=v....の文字列

sub MakeStrSETFromHashWithFilter {
	local($mode,*remove,*hash) = @_;
	local($k,$v);
 
	while(($k,$v) = each %hash) {
		if(&StringMatchToArray($k,@remove) eq "0") {
			$strtmp .= " $k = '$v',";
		}
	}
 
	chop($strtmp);
 
	$strtmp = "SET $strtmp";
 

	return $strtmp;
}
