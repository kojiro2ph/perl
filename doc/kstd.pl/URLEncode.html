URLEncode --- 文字列をURLエンコードする

渡し値(string)strtmp:エンコードする文字列
戻り値(string)strtmp:エンコードした文字列

sub URLEncode {
	local($strtmp) = @_;
	$strtmp =~ s/(\W)/'%'.unpack("H2", $1)/ego;
	return $strtmp;
}
