ListHash --- 連想配列をk=vにして配列に格納する

渡し値(integer)mode:オプション
(hash)hash:連想配列
戻り値(string array)k=vフォーマットの配列

sub ListHash {
	local($mode,%hash) = @_;
	local(@strarray);
 
	if($mode eq "1") {
		while(($k,$v) = each %hash) {
			push(@strarray,"$k=$v");
		}
 
		return @strarray;
	}
 
}
