ConvVal --- 文字列にある"_変数名_"をその変数名の値に変換する

渡し値(string)strtmp:変換する文字列
戻り値(string)strtmp:変換した文字列

sub ConvVal {
	local($strtmp) = @_;
		$strtmp =~ s/_(\w+)_/${$1}/g;
	return $strtmp;
}
