ConvPriceComma --- 数値にカンマをつける

渡し値(integer)カンマをつける数値
戻り値(integer)カンマつきの数値

sub ConvPriceComma {
    local($_) = @_;
    1 while s/(.*\d)(\d\d\d)/$1,$2/;
    $_;
}
